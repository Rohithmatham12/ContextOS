# Code of Conduct

## Our Pledge

We pledge to make participation in ContextOS welcoming, respectful, and
harassment-free for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but accept responsibility for your impact.
- Keep discussions focused on the project and its users.
- Prefer clear bug reports, reproducible examples, and actionable feedback.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks or sustained disruption.
- Publishing private information without permission.
- Any conduct that would make the project unsafe for contributors or users.

## Enforcement

Maintainers may remove comments, close issues, or block participants who violate
this code of conduct. Serious or repeated violations may result in a permanent
ban from project spaces.

## Scope

This code applies in all ContextOS project spaces, including GitHub issues,
pull requests, discussions, and any community channels associated with the
project.
